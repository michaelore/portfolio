Chemical Equation Balancer

This is a chemical equation balancing program made with Haskell. The parsing is done with Parsec, and the math is done with a small home-made linear algebra library. Command line and graphical interfaces are available.

Source

Balancer.hs Matrix.hs balancer-cli.hs balancer-gui.hs

Example

$ echo "H2 + O2 -> H2O" | ./balancer-cli
2H2 + O2 -> 2H2O
$ echo "CH4 + O2 -> CO2 + H2O" | ./balancer-cli
CH4 + 2O2 -> CO2 + 2H2O
